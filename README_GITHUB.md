# M.KULCZYK — publikacja GitHub Pages

## GitHub Pages
Wrzuć całą zawartość tego folderu do głównego katalogu repozytorium.

Strona online używa czystych adresów, np.:
- `/home/`
- `/projects/`
- `/articles/`
- `/articles-1/`
- `/loadplanner/`
- `/barometr/`

## Otwieranie lokalne
Dla lokalnego podglądu otwieraj pliki `home.html`, `projects.html`, `articles.html`, `loadplanner.html` itd.
Pliki `.html` zostają celowo, aby lokalny podgląd działał bez serwera.
